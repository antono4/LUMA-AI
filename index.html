🚀 Features

Everything You Need

Multiple ways to work with OpenHands - from SDK to cloud deployment

🧩

Software Agent SDK

Composable Python library with all agentic tech. Define agents in code, run locally or scale to 1000s in the cloud.

⌨️

CLI Interface

Terminal-based interface for running agents. Power it with Claude, GPT, or any other LLM.

🖥️

Local GUI

Browser-based UI with REST API and React SPA. Experience familiar to Devin or Jules.

☁️

OpenHands Cloud

Fully managed deployment with GitHub/GitLab/Bitbucket integrations, Slack, Jira, Linear, and collaboration features.

🏢

Enterprise

Self-host in your own VPC via Kubernetes. Source-available with extended support and research team access.

🎯

Skills System

Specialized prompts for domain-specific knowledge. GitHub, Docker, Security, Code Review, and more.
